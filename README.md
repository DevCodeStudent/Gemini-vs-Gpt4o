# Gemini vs GPT-4o: Full-Stack Dev Comparison

The **first developer-grade system**
comparing Google Gemini and OpenAI GPT-4o 
not just in features, but in how they're **built**.

Includes:
- Programming language + framework breakdown (JAX, TensorFlow vs PyTorch)
- Hardware differences (TPUs vs GPUs)
- Training infrastructure (Google Cloud vs Azure)
- API + ecosystem comparison
- Multimodal scope (Gemini 1.5 Pro vs GPT-4o)
- Openness, personality, and modularity traits

## Files Included

| File                    | Purpose                                               |
|-------------------------|-------------------------------------------------------|
| `gemini_vs_gpt4o.db`    | SQLite database of all core architectural differences |
| `gemini_vs_gpt4o.json`  | JSON spec for use in scripts, AI models, or dashboards|
| `gemini_vs_gpt4o.md`    | Markdown-formatted table for docs and GitHub display  |
| `gemini_vs_gpt4o.pdf`   | Printable PDF for educators, presentations, and reports|
| `gemini_vs_gpt4o.py`    | Python script to generate all above outputs           |
